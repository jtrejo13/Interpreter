# Interpreter
Simple mathematical expression interpreter implemented in C++

## Main classes:
* Token.cpp
* Scanner.cpp
* Parser.cpp
* Interpreter.cpp

## Examples of valid inputs:

```python
7 + 3 * (10 / (12 / (3 + 1) - 1))

7 + 3 * (10 / (12 / (3 + 1) - 1)) / (2 + 3) - 5 - 3 + (8)

7 + (((3 + 2)))
```
